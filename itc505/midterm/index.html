Murder Mystery Game

You wake up in a desolate world, surrounded by ruins. What will you do?

Addendum

The project consists of an HTML-based interactive storytelling game with associated CSS styling and JavaScript logic. The HTML file structures the game, linking to an external stylesheet for styling and an external script for game logic.

The CSS code focuses on creating a visually appealing and responsive user interface, while the JavaScript code defines the game's stages, choices, and consequences.

The user interface design emphasizes visual appeal through color schemes, button styling, and transitions. The JavaScript logic handles dynamic content with functions like updatePage and makeChoice, facilitating a structured and branching narrative. The use of images at each stage enhances the storytelling experience.

Challenges include managing dynamic content and narrative complexity, addressed through functions and a staged approach. Iterative development played a crucial role, allowing for adjustments based on testing feedback. Visual storytelling is a key insight, emphasizing the impact of images on user engagement in interactive narratives.

Overall, the project demonstrates a thoughtful and iterative development process, balancing complexity while prioritizing visual and interactive storytelling elements.
